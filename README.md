# Bankessentials

Bankessentials is a simple banking simulation app built with vanilla JavaScript.  
It allows users to log in, transfer money, request loans, and view transaction history with dates.

## Features

- Login/logout with username and PIN
- Deposits, withdrawals, transfers
- Loan requests (eligibility based on largest deposit)
- Transaction history with aligned dates
- Sorting of movements (ascending/descending)
- Balance and summary display (income, expenses, interest)
- Internationalized date and currency formatting

## Tech Stack

- HTML
- CSS
- JavaScript (ES2023 features like `toSorted`)
